# Glasses Images

Place your glasses PNG images here with transparent backgrounds:

1. **black-frames.png** - Black frame glasses (first image you provided)
2. **round-pink.png** - Round pink sunglasses (second image you provided)

## Requirements:
- PNG format with transparent background
- Front-facing view
- High resolution (recommended: 1000x400px or similar)

The glasses will be automatically scaled and positioned based on facial landmarks.
